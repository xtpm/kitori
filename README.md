# Kitori

## Run locally

```bash
npm install
npm run dev
```

Then open the local URL Vite prints, usually:

```bash
http://localhost:5173
```

## Build

```bash
npm run build
npm run preview
```

## Music files

Included track:
- Dancing with your eyes closed — Jane Remover

Place any future music files in `public/music/` and add them in `src/App.tsx`.
